# REST API for Car Service Center Appointment Management

## Technical stacks

Spring Boot

Maven

JPA

## Pre-reqs

Java, Maven (build tool)

## How to run test

./mvnw test
 
## How to start app

./mvnw spring-boot:run 

## How to build jar and start jar

./mvnw clean package

java -jar target/nielsen-sports-code-challenge-0.0.1-SNAPSHOT.jar
